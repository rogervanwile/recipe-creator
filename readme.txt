== Description ==

Create recipes and optimize them easily for search engines.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/foodblogkitchen-recipes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Screenshots ==

1. The recipe block in the Gutenberg editor.
2. Settings page to easily adjust the apperance of the recipe.
